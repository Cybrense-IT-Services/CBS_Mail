# Code of Conduct

## Our Pledge

We want CBS Mail to be a professional, respectful, and useful open-source
project. Contributors and users are expected to communicate with clarity,
patience, and respect.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent where possible.
- Keep issues focused on the project.
- Share enough context to reproduce bugs.
- Respect privacy and never post credentials, private mail content, or personal
  data in public issues.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information without permission.
- Posting production credentials, tokens, message contents, or user data.
- Spam, abuse, or intentionally disruptive behavior.

## Enforcement

Maintainers may edit, hide, or remove content that violates this code of
conduct. Repeated or serious violations may result in being blocked from the
project.

## Attribution

This is a lightweight project code of conduct written for CBS Mail. Projects
that need a more formal policy may adopt the Contributor Covenant.
