# Support

CBS Mail is an open-source Roundcube UI overlay. Support is community-based
unless a separate commercial agreement exists.

## Before Opening An Issue

Please check:

- The README setup steps.
- `CONTRIBUTING.md` verification commands.
- Browser console errors.
- Roundcube container logs.
- Whether the issue also happens in plain Roundcube.

## Good Bug Reports Include

- What you expected.
- What happened.
- Steps to reproduce.
- Browser and device details.
- Roundcube Docker image/tag.
- CBS Mail commit hash.
- Screenshots for UI bugs, with private data removed.

## Security Reports

Do not open public issues for exploitable security problems. See
`SECURITY.md`.

## Branding Questions

The Cybrense/CBS branding assets are included for this project. Forks for other
organizations should replace the logos, product name, PWA metadata, and default
trusted sender/domain settings.
