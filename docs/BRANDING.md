# Branding Guide

CBS Mail ships with Cybrense/CBS branding so the default interface works out of
the box.

```text

```

## Website

Official Cybrense website:

```text
https://cybrense.com/
```

## Forking

If you fork CBS Mail for another organization, replace:

- logos,
- favicons,
- PWA icons,
- manifest name and colors,
- product name,
- default trusted sender/domain values,
- README header artwork.
